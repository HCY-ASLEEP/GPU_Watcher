# GPU_Watcher
Alternative of nvitop.<br>
Faster than nvitop, lighter than nvitop too, but similar to nvitop capacities.<br>
Refresh per second defaultly.

## Usage
```bash
python gpu_watcher.py
```

Ctrl-C to exit.
